# Shein申报价异常V1

更新时间：2026-06-03

## 目标

生成 Shein 申报价异常交叉对比表，用于按店铺识别：

- Shein 供货价/采购价低于 ERP 成本价的亏损产品。
- Shein 供货价/采购价低于 ERP 批发价 80% 的破价产品。
- 各店铺在售 SKC、7 天销量、30 天销量、30 天内有销量的 SKC 数、爆旺款、亏损 SKC、破价 SKC 概况。

## 数据源

- Shein 备货单导出：`shein数据源表/20260601-shein琪琪.xlsx`
- Shein 备货单导出：`shein数据源表/20260601-shein童话.xlsx`
- Shein 备货单导出：`shein数据源表/20260601-shein加加.xlsx`
- Shein 备货单导出：`shein数据源表/20260601-shein宝宝.xlsx`
- Shein 备货单导出：`shein数据源表/20260601-shein牛牛.xlsx`
- ERP 基础信息：`erp数据源/erp产品基础信息表*.xlsx`
- ERP 组合装基础信息：`erp数据源/erp产品组合装基础信息表.xlsx`
- 店铺负责人：`店铺负责人对应表.xlsx`

## 字段映射

### Shein 字段

- 店铺：从源文件名识别，固定为 `琪琪`、`童话`、`加加`、`宝宝`、`牛牛`
- SKC：`SKC`
- 商家编码 / SKU：`商家SKU`
- 申报价：优先取 `供货价`，没有时取 `采购价`
- 7 天销量：`近7天销量`
- 30 天销量：`近30天销量`
- 爆旺款识别辅助字段：`商品标签`

### ERP 字段

- 商家编码：`商家编码（新）`
- 货品名称：`货品名称`
- 货品规格：`规格名称`
- 成本价：`成本价`
- 批发价：`批发报价`

### 组合装字段

- 若 Shein `商家SKU` 无法直接匹配 ERP 基础信息，则继续匹配 ERP 组合装基础信息。
- 组合装成本价和批发价以组合装基础信息表为准，不留空。

## 匹配规则

- Shein `商家SKU` 对应 ERP `商家编码（新）`。
- 若编码含 `@`，匹配时忽略 `@` 及后续字符。
- 输出明细中的商家编码使用 ERP 匹配后的 `商家编码（新）` 或组合装编码。
- 完全无法匹配 ERP 基础信息或组合装基础信息的行，不进入异常明细，只在数据校验中统计跳过数量。

## 店铺顺序

按以下固定顺序输出：

```text
琪琪 -> 童话 -> 加加 -> 宝宝 -> 牛牛
```

## 预警定义

### 亏损产品

条件：

```text
Shein 申报价 < ERP 成本价
```

说明：

- Shein 申报价 = `供货价`，没有 `供货价` 时使用 `采购价`。
- 命中后输出到 `低于成本价` 页签。

### 破价产品

条件：

```text
Shein 申报价 < ERP 批发价 * 80%
```

说明：

- 命中后输出到 `低于批发价80%` 页签。
- 该规则用于识别明显低于批发报价体系的产品。

### 爆旺款数量

当前 V1 口径：

```text
商品标签包含“高销款”“爆”“旺”任一关键词
```

按店铺统计唯一 SKC 数。

## 总览表口径

总览表按店铺输出以下字段：

- 店铺
- 负责人
- 在售 SKC 数量：该店铺 Shein 源表中唯一 `SKC` 数量
- 7 天销量：该店铺 `近7天销量` 汇总
- 30 天销量：该店铺 `近30天销量` 汇总
- 30 天内有销量的 SKC 数：该店铺 `近30天销量 > 0` 的唯一 `SKC` 数量
- 爆旺款数量：按当前 V1 爆旺款口径统计唯一 `SKC` 数量
- 亏损 SKC 数量：该店铺命中 `低于成本价` 的唯一 `SKC` 数量
- 破价 SKC 数量：该店铺命中 `低于批发价80%` 的唯一 `SKC` 数量

## 排序规则

异常明细按以下顺序排序：

1. 第一排序：店铺，顺序为 `琪琪 -> 童话 -> 加加 -> 宝宝 -> 牛牛`
2. 第二排序：`SKC`
3. 第三排序：`商家编码`
4. 商家编码尺码顺序：

```text
XXS -> XS -> S -> M -> MD -> L -> XL -> XXL
```

同一店铺、同一 SKC、同一款不同尺码时，按上述尺码顺序排列。

## 输出文件

命名规则：

```text
yymmdd-Shein申报价异常-版本号.xlsx
```

当前示例：`260603-Shein申报价异常-V1.xlsx`

输出页签：

- `总览表`
- `低于成本价`
- `低于批发价80%`
- `数据校验`
