# ERP Settings Page Design

## Goal

ERP 接口设置从弹窗改为管理员专用二级页面，避免同步异常显示在背后页面导致看不见。

## Design

系统设置页保留“ERP 接口设置”卡片。管理员点击后进入独立的 `ERP 接口设置` 页面，左上区域展示返回系统设置按钮。

页面分为两块：

1. 接口配置：接口服务商、环境、接口地址、SID、AppKey、AppSecret、仓库编码、仓库名称、同步天数、每页条数、库存最多拉取。
2. 同步内容和结果：商品基础资料、仓库库存、平台货品、销售出库单、启用接口、自动同步、手动同步为主，以及保存、同步、本地校验按钮。

不常用字段放入高级设置，默认收起：Token、店铺 ID、店铺编码、接口文件名、可用库存、店铺接口、同步范围。

## Error Handling

同步失败时，结果显示在当前页面下方，不再依赖弹窗或背后页面。主提示显示人能看懂的原因，技术详情默认折叠。

只同步商品基础资料和仓库库存时，不要求填写店铺 ID。只有勾选销售出库单或店铺查询时，才要求店铺 ID 或店铺编码。
